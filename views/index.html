Timestamp Microservice

A little Node/Express API that converts UTC date to UNIX timestamp and UNIX timestamp to UTC date

How to use it :

  • Make a request to /api/ route with a UTC date or a UNIX timestamp as the URL [param-to-convert] : 

    Example of usage:
https://baseUrl/api/[param-to-convert]

Example Usage :

  • With a date string URL param :

    https://node-express-timestamp-microservice.sebastiennoret.repl.co/api/2015-12-25

  • With a timestamp string URL param :

    https://node-express-timestamp-microservice.sebastiennoret.repl.co/api/1451001600000

Example Output :

{ 
    "unix" : 1660003200000,
    "utc" : "Tue, 09 Aug 2022 00:00:00 GMT"
}

Error Output:

{ 
    "error" : "Invalid Date"           
}
