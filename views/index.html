Timestamp Microservice

A little node API that returns a timestamp for a given URL param date string
And a UTC date string for a given timestamp URL param

Example Usage:

  • With a date string URL param:

    [project url]/api/2015-12-25

  • With a timestamp string URL param:

    [project url]/api/1451001600000

Example Output:

With a date string URL param:

{"unix":1660003200000,"utc":"Tue, 09 Aug 2022 00:00:00 GMT"}

With a timestamp string URL param:

{"unix":1451001600000, "utc":"Fri, 25 Dec 2015 00:00:00 GMT"}
